## 功能特性

### 🔍 智能列名识别
不同风场的 SCADA 导出文件列名往往各不相同（如"发电机转速" / `Gen_RPM` / `rotspeed`）。工具内置语义槽匹配引擎，覆盖中英文、缩写、带单位等多种命名方式，自动将 CSV 列映射到以下 8 个标准信号槽：

| 信号槽 | 匹配示例 |
|--------|---------|
| 有功功率 | `发电机有功功率` / `Active Power` / `kW` / `P_gen` |
| 转速 | `发电机转速` / `RPM` / `Gen_Speed` / `r/min` |
| 风速 | `风速` / `Wind Speed` / `ws` / `v_wind` |
| 风向 | `风向` / `Wind Direction` / `wind_dir` |
| 桨距角 1/2/3 | `1#桨叶片角度` / `Pitch1` / `Blade_1` / `PA1` |
| 齿轮箱温度 | `齿轮箱油池温度` / `Gearbox Temp` / `GB_Temp` |

每次加载文件时弹出**确认对话框**，可人工校正自动识别结果，确保映射准确。

### 📂 灵活的数据加载
- **单文件加载**：支持任意编码的单个 CSV 文件
- **批量加载**：一次选择多个 CSV，自动纵向拼接
- **自动编码探测**：依次尝试 `UTF-8` / `GBK` / `GB2312` / `GB18030` 等编码，兼容国内风场常见的 GBK 格式，无需手动指定

### 🪟 滑动窗口分割
- 可自由设置窗长 `W`（时间步数）和步长 `S`
- 实时显示分窗数量，方便调整参数

### 📊 多通道可视化
- 每个识别到的信号独占一行子图，颜色固定便于区分
- 每张子图显示均值线（金色虚线）及 avg / min / max 统计数据
- 支持 Matplotlib 原生缩放、平移工具栏
- 可勾选/取消勾选任意通道

### 🏷️ 标注功能
- **风速等级**：低 / 中 / 高 / 超高风速，自动从风速信号推断并预填
- **故障/状态类型**：9 种预设类型（正常运行、变桨异常、齿轮箱故障等）
- **自定义标签**：输入任意文本，覆盖预设组合
- **备注**：为每条标注附加文字说明
- **历史标签**：双击已使用过的标签快速复用
- **跳过**：对不确定片段打跳过标记，不计入最终数据集

### ⌨️ 快捷键
| 按键 | 动作 |
|------|------|
| `←` / `→` | 上一个 / 下一个片段 |
| `Enter` | 确认标注 |
| `S` | 跳过当前片段 |
| `Delete` | 删除当前片段的标注 |

## 使用说明

### 1. 加载数据

点击 **📂 加载 CSV** 选择单个文件，或 **📁 批量加载** 一次选择多个文件。

弹出的**列名识别对话框**会显示自动匹配结果：
- ✅ 绿色：已成功识别，显示对应的原始列名
- ❌ 橙色：未识别，可通过下拉菜单手动指定

确认后工具开始分窗并显示第一个片段。

### 2. 设置分窗参数

在工具栏调整 **窗长 W**（每段包含的时间步数）和 **步长 S**（相邻窗口的偏移量），点击 **⚙ 应用分窗** 后重新加载文件生效。

### 3. 标注片段

在右侧面板：
1. 选择**风速等级**（工具会根据风速信号自动推断并预选）
2. 选择**故障/状态类型**
3. 如需自定义，在"自定义标签"框中输入文本（留空则自动组合风速+故障类型）
4. 可选填**备注**
5. 点击 **✅ 确认标注** 或按 `Enter`

勾选"标注后自动跳转"可在确认后自动前进到下一个片段。

### 4. 保存结果

点击 **💾 保存结果**，选择保存路径，输出 `.npy` 文件。同时生成 `label_db.json` 保存历史标签记录。

## CSV 格式要求

- 文件格式：逗号分隔的 `.csv` 文件
- 编码：自动识别，支持 UTF-8、GBK、GB2312、GB18030 等
- 每行代表一个时间采样点，每列代表一个信号
- 允许包含时间列（`时间` / `Time` / `Timestamp`），工具会自动忽略
- 缺失值支持：`N/A`、空格、`nan`、`NaN`

**列名无强制要求**，工具通过关键词模糊匹配自动识别。以下为两种命名风格的示例：

```
# 中文风场导出格式
时间,发电机有功功率,发电机转速,风速,风向,1#桨叶片角度,2#桨叶片角度,3#桨叶片角度,齿轮箱油池温度

# 英文/混合格式
Timestamp,Active_Power_kW,Gen_RPM,Wind_Speed,Wind_Dir,Pitch1_deg,Pitch2_deg,Pitch3_deg,GB_OilTemp
```

## 输出文件说明

### `labeled_dataset.npy`

使用 `np.load(..., allow_pickle=True)` 加载，返回一个字典：

```python
import numpy as np

data = np.load('labeled_dataset.npy', allow_pickle=True).item()
# data 结构: { '标签名': np.ndarray(shape=(N, W, D)), ... }
# N = 该标签下的片段数量
# W = 窗长（时间步数）
# D = 识别到的信号维度数

for label, arr in data.items():
    print(f"{label}: {arr.shape}")
# 示例输出:
# 低风速(0-5m/s)_正常运行: (42, 10, 5)
# 中风速(5-12m/s)_变桨异常: (8, 10, 5)
```

信号维度的顺序与加载时确认的列映射顺序一致（按 `SLOT_ORDER` 排列，未识别的槽不包含在内）。

### `label_db.json`

记录所有历史使用过的标签及其使用次数和备注，用于跨会话复用。格式示例：

```json
{
  "低风速(0-5m/s)_正常运行": { "count": 42, "remark": "" },
  "中风速(5-12m/s)_变桨异常": { "count": 8, "remark": "叶片角度偏差>3°" }
}
```

## 扩展与定制

### 添加新的故障类型

修改脚本中的 `PRESET_FAULT_TYPES` 列表：

```python
PRESET_FAULT_TYPES = [
    '正常运行', '变桨异常', '齿轮箱故障', '发电机故障',
    '振动异常', '超速保护', '低功率异常', '温度异常', '其他故障',
    '自定义类型1',   # ← 在此添加
]
```

### 扩展信号槽的匹配关键词

在 `SEMANTIC_SLOTS` 字典中为对应槽追加关键词：

```python
'转速': {
    'keywords': [..., '你的自定义列名'],
    ...
},
```

### 修改风速等级划分

```python
WIND_LEVELS = {
    '低风速':  (0, 5),
    '中风速':  (5, 12),
    '高风速':  (12, 99),
}
```
